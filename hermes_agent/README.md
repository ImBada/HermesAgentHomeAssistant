# Home Assistant Add-on: Hermes Agent

Run Hermes Agent as a supervised Home Assistant app.

Hermes can subscribe to selected state changes, reason over them with an LLM, call Home Assistant services through its built-in tools, and send replies back as persistent notifications.

This add-on includes a Home Assistant Ingress page, an optional Hermes dashboard, and an optional web terminal.

See [DOCS.md](DOCS.md) for configuration details.
